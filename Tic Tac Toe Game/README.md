# 🎮 Tic Tac Toe Game

A modern and interactive **Tic Tac Toe Game** developed using **HTML5, CSS3, and JavaScript** with a clean UI, responsive layout, animated effects, player statistics tracking, and popup notifications.

This project provides a stylish multiplayer gaming experience where two players can enter their names, play multiple matches, track wins and draws, and enjoy smooth gameplay with a visually appealing interface.

* 🎯 Two-player Tic Tac Toe gameplay

# 🎮 How to Play

1. Enter Player 1 and Player 2 names.
2. Click the **Start Game** button.
3. Players take turns placing:

   * `X` for Player 1
   * `O` for Player 2
4. The first player to align:

   * 3 rows
   * 3 columns
   * or 3 diagonally
     wins the game.
5. If all cells are filled without a winner, the match ends in a draw.
6. Statistics update automatically after every match.
